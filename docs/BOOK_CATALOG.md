# Catálogo inicial de livros — Learning Paths

Formato de seed para a tabela `books`. Cada entrada deve virar uma linha
com `pattern_id`, `level_range` e `tags`.

## Notas de curadoria

- Não sugerir mais de 1 livro do mesmo autor em sequência (diversidade de vozes).
- Priorizar disponibilidade em português quando possível; registrar
  formato (ebook/físico/audiobook) por edição.
- Este catálogo é o seed inicial — deve crescer via tabela `books`, não
  hardcoded no frontend, para permitir curadoria contínua sem redeploy.
